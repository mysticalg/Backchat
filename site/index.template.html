What’s inside

Built for private conversation, not just polished screenshots.

Focused desktop experience

Contact-side unread counters, taskbar badges, quick contact switching, and conversation focus that feels like a real desktop messenger.

Local-first history

Conversation logs are cached per machine so your message history and unread state stay available locally without turning the server into your archive.

Presence and profiles

Contacts can see avatar, quote, and online state at a glance so the contact list works like a useful dashboard, not just a list of names.

Direct-friendly calling

Voice and video calls use WebRTC signaling with advanced routing controls for auto, direct/VPN-preferred, direct-only, or relay-only behavior.

Security snapshot

Practical encryption with room to grow.

Backchat already ships encrypted-message primitives, presence, and direct-call signaling. The project is structured so you can keep strengthening the transport and device story over time.

Encrypted messaging primitives

X25519 key exchange and AES-GCM are already in the app scaffold for message protection.

Direct media when possible

Voice and video attempt peer-to-peer routing first, with relay fallback as an option when network conditions need it.
